## 一、项目概述

### 1.1 项目名称与目标

| 项 | 说明 |
|---|---|
| 项目名 | **SHOO** |
| 类型 | 跨平台电商移动客户端（iOS / Android） |
| 核心目标 | 构建接近 Shein 体验的完整电商客户端 |
| 参考对象 | **Shein**（主视觉）、**淘宝**（营销）、**京东**（订单/物流） |
| 交付形态 | 分阶段迭代，每阶段可运行可演示 |

### 1.2 非目标

- 不自研后端（Mock 为主）
- 首期不做商家端 / 运营后台
- 首期不做直播带货
- 真支付放到 Phase 4，首期模拟支付

---

## 二、功能清单（摘要）

完整功能列表见 v1.0 方案。Phase 1 优先：

## 三、框架设计（当前实现）

```
lib/
├── app/                    # 入口、路由、Tab Shell
├── core/
│   ├── constants/
│   ├── errors/
│   ├── network/            # Dio + MockInterceptor
│   ├── storage/
│   ├── theme/              # Shein 风格 Token
│   ├── utils/
│   └── widgets/            # 基础组件库
└── features/
    ├── home/
    ├── category/
    ├── cart/
    └── profile/
```

**技术栈**：Riverpod · go_router · Dio · shared_preferences · cached_network_image
